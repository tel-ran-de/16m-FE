Описание некоторых тегов

abbr
Тег показывает, что текст внутри тега является аббревиатурой
Написание

<abbr>Аббревиатура</abbr>

Добавление title дает всплывающую подсказку расшифорвки аббревиатуры

< abbr title="Расшифровка">Аббревиатура</abbr>

Пример:

< abbr title="National Aeronautics and Space Administration">NASA</abbr>

Отображение: NASA
address
Определяет содержимое тега как контактную информацию. Исользование тега <adress> внутри тега <article> определяет принадлежность контактной ифнормации к автору статьи. В другом случае: контактная информация относится к странице в целом.
Написание

<address>Контактная информация</address>

Пример:

<address> Вы можете написать мне <a href="mailto:mymail@mymail.com">Имя  
или <a href="www.mysite.my">Мой сайт для связи </address>

Отображение:
Вы можете написать мне Имя
Мой сайт для связи

aside
Появляется блок сбоку от контента, с неосновным содержимым, например дополнительные комментарии, справочная информация
Написание

<aside> Текст </aside>

Пример:

<aside> Lorem ipsum, dolor sit amet consectetur adipisicing elit. Obcaecati magnam repellendus eum animi non illo, rem officiis dignissimos beatae voluptatum adipisci fugit, numquam veniam laborum dolor voluptate porro quam ducimus. </aside>

Отображение:
Lorem ipsum, dolor sit amet consectetur adipisicing elit.

code
Используется для отображения программного кода. Обычно отображается уменьшенным шрифтом
Написание

<code>программный код </code>

Пример:

<code>for (int i = left; i < right; i++) { if (books[i].price> books[i + 1].price) </code>

Отображение:
for (int i = left; i < right; i++) { if (books[i].price > books[i + 1].price)
caption
Определяет заголовок или название таблицы, поэтому размещается только внутри <table> По умолчанию, заголовок выводится над таблицей и выравнивается по центру.
Написание

<caption>Заголовок </caption>

Пример:

<table border="1"> <caption>Времена года</caption> <tr> <th>Месяц</th> <th>Время года</th> </tr> <tr> <td>Декабрь</td> <td>Январь</td> </tr> </table>

Отображение:
Времена года
Месяц Время года
Декабрь Зима
cite
Предназначен для выделения названия работы (например, книг,веб-сайтов, песен, фильмов, живописи и тд). Текст помечается курсивным начертанием.
Написание

<cite>текст</cite>

Пример:

<cite> «Дюна» </cite> — один из самых известных научно-фантастических романов XX века

Отображение:
«Дюна» — один из самых известных научно-фантастических романов XX века
fieldset
Контейнер для других элементов, при отображении в браузере по умолчанию он выделяется рамкой и имеет небольшой внутренний отступ со всех сторон. Обычно его используют для визуального выделения группы логически связанных между собой элементов формы.
Написание

<fieldset>текст</fieldset>

Пример:

<form>
<fieldset>
< legend>Регистрация:</legend>
Имя: <input type="text"><br>
Фамилия: <input type="text"><br>
e-mail: <input type="email">
</fieldset>
</form>

Отображение:
Регистрация: Имя:
Фамилия:
e-mail:

<figure> <img src="iguana.jpg" alt="Iguana" width="161" height="100"> <figcaption>Это игуана.</figcaption> </figure>

Написание

<legend>текст</legend>

Пример:

<form>
<fieldset>
< legend>Регистрация:</legend>
Имя: <input type="text"><br>
Фамилия: <input type="text"><br>
e-mail: <input type="email">
</fieldset>
</form>

Отображение:
Регистрация: Имя:
Фамилия:
e-mail:
mark
Помечает текст как выделенный. Такой текст ничем не отличается от обычного, но его вид может быть изменен с помощью стилей. В браузере Chrome и Firefox фоновый цвет текста внутри <mark> выделяется желтым цветом.
Написание

<mark>текст</mark>

Пример:

<mark> «Дюна» </mark> — один из самых известных научно-фантастических романов XX века

Отображение:
«Дюна» — один из самых известных научно-фантастических романов XX века

optgroup
Представляет собой контейнер, внутри которого располагаются теги <option> и используется для группировки связанных логически вариантов в выпадающем списке. Тег <optgroup> не выделяется, как обычный элемент списка, а выглядит как заголовок жирного начертания. Все элементы (тег <option>), входящие в этот контейнер, смещаются немного вправо от своего исходного положения.
Пример:

<select>
<optgroup label="Цикл Dune">
<option value="Dune">Dune</option>
<option value="Dune Messiah">Dune Messiah</option>
<option value="Children of Dune">Children of Dune</option>
</optgroup>
</select>

Отображение:

samp
Определяет текст как результат вывода компьютерной программы.
Браузеры обычно отображают текст моноширинным шрифтом.
Помимо тега <samp> в HTML используются другие теги логического форматирования текста, связанные с компьютерным вводом: <code> - код компьютерной программы, <kbd> - название клавиш на клавиатуре, <var> - переменная компьютерной программы.
Написание

<samp>текст</samp>

Отображение:
Pезультат вывода компьютерной программы: Hello World

th
Тег <th> предназначен для создания одной ячейки таблицы, которая обозначается как заголовочная. Текст в такой ячейке отображается браузером обычно жирным шрифтом и выравнивается по центру. Тег <th> должен размещаться внутри контейнера <tr> , который в свою очередь располагается внутри тега <table> .
Написание

< table>
<tr>
<th>...</th>
</tr>
</table>

thead
Предназначен для хранения одной или нескольких строк, которые представлены вверху таблицы. Допустимо использовать не более одного элемента в пределах одной таблицы, и он должен идти в исходном коде сразу после тега <table> .
Написание

<table>
<thead>
<tr>
<td> ... </td>
</tr>
</thead>
<tfoot> ... </tfoot>
<tbody> ... </tbody>
</table>

meter
Используется для вывода значения в некотором известном диапазоне. Используется преимущественно для отображения числовых значений, например, количества результатов поиска, объема жидкости, давления и др.
Атрибуты

  • value Устанавливает значение. Обязательный атрибут.
  • min Задает минимально возможное значение.
  • max Задает максимально возможное значение.
  • low Определяет предел, при достижении которого значение считается низким
  • high Определяет предел, при достижении которого значение считается высоким.
  • optimum Определяет наилучшее или оптимальное значение.

Пример:

<meter value="2" min="0" max="10">2 из 10</meter>
<meter value="0.6">60%</meter>

Отображение:
2 из 10
60%

keygen
Используется для генерации пары ключей — закрытого и открытого для форм. Сами ключи необходимы для шифрования и расшифровки данных. При отправке формы, закрытый ключ сохраняется на локальном компьютере, а открытый ключ передается на сервер вместе с формой.
Написание Пример:

<input type="submit">
</form>

Имя пользователя: Шифрование:
main
Основной контент. Предназначен для основного содержимого документа. Содержимое должно быть уникальным и не включать типовые блоки вроде шапки сайта, подвала, навигации, боковой панели, формы поиска и т. п, например: меню, информацию об авторских правах или форму поиска.
Написание

<main>текст</main>

embed
Контейнер для внешнего приложения, используется для загрузки и отображения объектов (например, видеофайлов, флэш-роликов, некоторых звуковых файлов и т.д.), которые исходно браузер не понимает. Как правило, такие объекты требуют подключения к браузеру специального модуля, который называется плагин, или запуска вспомогательной программы.
Написание

<embed width="..." height="..."></embed>
